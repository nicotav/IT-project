# IT Level 2 Support Engineer Portfolio Plan

## Recommended Tools

1. **Scripting**: PowerShell (Windows), Bash (Linux), Python (cross-platform)
2. **Version Control**: Git, GitHub
3. **Documentation**: Markdown (README.md), MkDocs or Sphinx (optional)
4. **Automation**: Task Scheduler (Windows), cron (Linux), GitHub Actions (CI/CD)
5. **Monitoring**: Built-in tools (Performance Monitor, top/htop), Python psutil
6. **Web Development**: Flask (Python), Node.js/Express, HTML/CSS/JS
7. **Log Analysis**: Python (regex, pandas), PowerShell
8. **Remote Management**: PowerShell Remoting, SSH, paramiko (Python)

## Portfolio Project Ideas & Estimated Time

1. **Automated System Health Check Script** (2–4 hours)
2. **Ticketing System Simulation** (4–8 hours)
3. **Log File Analyzer** (2–4 hours)
4. **Backup and Restore Automation** (2–3 hours)
5. **Remote Software Deployment Script** (2–3 hours)
6. **Network Troubleshooting Toolkit** (2–3 hours)
7. **Documentation Example** (1–2 hours)
8. **Basic Monitoring Dashboard** (4–6 hours)

**Total Estimate:** 19–33 hours (3–5 days)
